| 手势 | 场景 | 效果 |

### 5.2 PC端快捷键

| 快捷键 | 功能 |
|-------|------|
| Ctrl + R | 刷新数据 |
| Ctrl + E | 导出当前页 |
| Ctrl + F | 搜索 |
| ESC | 关闭弹窗 |

### 5.3 通用交互

- 所有操作按钮需提供 loading 状态
- 表单提交失败需显示具体错误信息
- 列表数据为空需显示空状态插图
- 网络异常需提示并支持重试
