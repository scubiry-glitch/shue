## 3. 服务划分

### 3.1 考勤服务 (Attendance Service)
- **端口**: 3001
- **职责**: 核心打卡逻辑、NFC验证、GPS校验
- **核心模块**:
  - NFC读取验证
  - 打卡记录CRUD
  - GPS距离计算
  - 质量评分计算

### 3.2 绩效计算服务 (Performance Service)
- **端口**: 3002
- **职责**: 绩效指标实时计算、排名更新
- **核心模块**:
  - 系数A/B/C计算
  - 百分位排名
  - 异常检测
  - 数据同步

### 3.3 消息推送服务 (Notification Service)
- **端口**: 3003
- **职责**: 飞书消息推送、短信通知
- **核心模块**:
  - 消息模板管理
  - 推送队列处理
  - 消息状态追踪

### 3.4 房源服务 (House Service)
- **端口**: 3004
- **职责**: 房源信息管理、NFC标签关联
- **核心模块**:
  - 房源CRUD
  - NFC标签绑定
  - 位置信息管理

### 3.5 用户管理服务 (User Service)
- **端口**: 3005
- **职责**: 用户管理、权限控制、设备绑定
- **核心模块**:
  - 用户CRUD
  - 角色权限
  - 设备绑定管理

### 3.6 飞书集成服务 (Feishu Service)
- **端口**: 3006
- **职责**: 飞书开放平台对接
- **核心模块**:
  - 身份认证
  - 消息卡片
  - 审批流
  - 多维表格同步

## 6. 安全设计

### 6.1 网络安全
- 全站HTTPS (TLS 1.3)
- API Gateway IP白名单
- DDoS防护
- WAF规则防护

### 6.2 应用安全
- JWT Token认证
- API限流 (100次/分钟)
- 敏感数据脱敏
- SQL注入防护

### 6.3 数据安全
- 数据库连接加密
- 敏感字段加密存储
- 定期备份 (每日全量+实时增量)
- 数据保留策略 (90天)
